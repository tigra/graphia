# Tasks: Active Scripted Player for Measured Runs (Spec 026)

Eval-only, deterministic policy (no LLM, no RNG). Gated by a **default-on** boolean flag `GRAPHIA_ACTIVE_SCRIPTED_PLAYER` (active by default; per ADR 011) — `--scripted-player passive` reproduces the prior baseline. The policy itself is precisely **unit-testable** (deterministic); only the downstream win-rate effect is **effort-not-results** (CR 005).
