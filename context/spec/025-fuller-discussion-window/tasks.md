# Tasks: Fuller Multi-Day Discussion Window for AI Players (Spec 025)

Gameplay-influencing → the window length is a **default-on tunable** (`GRAPHIA_CONTEXT_WINDOW`, default ~150 ≈ 3+ days) so the prior 30 is reproducible for A/B (ADR 011). Acceptance is **effort-not-results** (CR 005). **Ollama context route = A (decided):** keep `ChatAnthropic`; the fuller window is delivered by the operator's server-side `OLLAMA_CONTEXT_LENGTH=32768` (already set), backed by an app-side token-budget cap + a startup context check — no ADR-010 change.
