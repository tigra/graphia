# Tasks: Continuous Persona-Similarity Metrics (Average + Peak) (Spec 032)

Two vertical slices: **Slice 1** records *and* displays the average + peak lexical persona similarity (the smallest end-to-end visible increment); **Slice 2** backfills the transcript-preserved past runs. Follow-up to **spec 031 (Distinct AI Personas)**; pure-Python, lexical/deterministic, additive — **no `METRICS_VERSION` bump, no ablation flag** (measurement-only, ADR-011 / architecture §6 exempt).

> **Shared surface with spec 033 (semantic):** this spec introduces the reusable **value-type metric shape**, the **value-type viewer render**, and the **transcript backfill harness** that 033 reuses. **Land 032 before 033.**
